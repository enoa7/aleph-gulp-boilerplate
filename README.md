# Aleph FE Gulp Boilerplate
## Pre-requisite
* NodeJs
* NPM
* NPX (optional)
* Yarn
* Gulp 4 (installed globally)

## How To
* run `yarn install`
* open terminal and run `gulp` or run `yarn build`. This will compile Pug, SCSS and JS files to build folder
